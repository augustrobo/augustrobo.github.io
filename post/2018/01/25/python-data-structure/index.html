Built-in

List

  • See help(list)
# Create a new list
empty = list()
# or
empty = [] # faster, more pythonic
  • Lists can contain elements of different types
  • Use append to append elements to the end of a list
nums = [1,2,3]
nums.append(4) # nums == [1, 2, 3, 4]
  • Access element at a particular index
nums[-1] # => [4]
  • Slice lists
nums[1:-1] # => [2, 3]
'python'[:3] # => 'pyt'
  • Nested lists
letters = ["a", "b", "c"]
x = [nums, letters]
x[0] # => [1, 2, 3, 4]
x[0][1] # => 2
x[1][:2] # => ['a', 'b']
  • Length (len)
len(x) # => 2
len([]) # => 0
  • Membership (in)
0 in [] # => False
'y' in 'python' # => True
  • Palindrome?
# Palindrome Check
word = input("Please enter a word: ")
print(word == word[::-1])
  • Check for an empty list
data = []
if data:
    process(data):
else:
    print("There's no data!")
  • for loops
for item in iterable: # strings, lists, etc.
    process(item)
