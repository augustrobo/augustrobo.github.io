Maximum Subarray

Find the contiguous subarray within an array (containing at least one number) which has the largest sum.

For example, given the array [-2,1,-3,4,-1,2,1,-5,4], the contiguous subarray [4,-1,2,1] has the largest sum = 6.

Merge Sorted Array

Given two sorted integer arrays nums1 and nums2, merge nums2 into nums1 as one sorted array.

Note: You may assume that nums1 has enough space (size that is greater or equal to m + n) to hold additional elements from nums2. The number of elements initialized in nums1 and nums2 are m and n respectively.

Non-decreasing Array

Given an array with $n$ integers, your task is to check if it could become non-decreasing by modifying at most 1 element.

We define an array is non-decreasing if array[i] <= array[i + 1] holds for every i (1 <= i < n).

思路:

使用for loop扫描数列,假如在下标idx, idx+1处不满足非递减关系,那么:

  • 要么 array[idx-1]<=array[idx+1],此时只要判断idx+1后都满足非递减关系即可
  • 要么array[idx-1]>array[idx+1],此时只要用array[idx]代替array[idx+1],再判断idx+1后都满足非递减关系即可
class Solution:
    def checkPossibility(self, nums):
        """
        :type nums: List[int]
        :rtype: bool
        """
        idx = 0
        for i in range(len(nums)-1):
            if nums[i] > nums[i+1]:
                idx = i
                break

        if idx > 0 and (nums[idx-1] > nums[idx+1]):
            nums[idx+1] = nums[idx]

        for j in range(idx+1, len(nums)-1):
            if nums[j] > nums[j+1]:
                return False
        else:
            return True

Two Sum

Given an array of integers, return indices of the two numbers such that they add up to a specific target.

You may assume that each input would have exactly one solution, and you may not use the same element twice.

方式一：Brute Force

class Solution:
    def twoSum(self, nums, target):
        for idx1, element in enumerate(nums):
            try:
                idx2 = nums[idx1 + 1:].index(target - element) + idx1 + 1
            except ValueError:
                continue
            return [idx1, idx2]
        return None
  • Time complexity: $O(n^2)$
  • Space complexity: $O(1)$

方式二：One-pass Hash Table

这里有个特别聪明的做法，就是先查找，再插入

class Solution:
    def twoSum(self, nums, target):
        numDict = {}
        for idx1, element in enumerate(nums):
            try:
                idx2 = numDict[target - element]
            except KeyError:
                numDict[element] = idx1
                continue
            return [idx1, idx2]
        return None
  • Time complexity: $O(n)$
  • Space complexity: $O(n)$

方式三：先排序

  • 排序，$O(n\log n)$
  • 然后对每个 i，在排序后的 nums 中二分查找 target - nums[i]，$n\times O(\log n)=O(n\log n)$
  • 因此，$T(n) = O(n\log n)$

Find Peak Element

A peak element is an element that is greater than its neighbors.

Given an input array where num[i] ≠ num[i+1], find a peak element and return its index.

The array may contain multiple peaks, in that case return the index to any one of the peaks is fine.

You may imagine that num[-1] = num[n] = -∞.

For example, in array [1, 2, 3, 1], 3 is a peak element and your function should return the index number 2.

class Solution:
    def findPeakElement(self, nums):
        """
        :type nums: List[int]
        :rtype: int
        """
        # base cases
        if len(nums) == 1:
            return 0
        elif len(nums) == 2:
            return int(nums[0]<nums[1])

        mid = len(nums)//2
        if nums[mid] < nums[mid-1]:
            return self.findPeakElement(nums[:mid])
        elif nums[mid] < nums[mid+1]:
            return self.findPeakElement(nums[(mid+1):]) + mid + 1
        else:
            return mid

$T(n)=O(\log n)$

Excel Sheet Column Title

Given a positive integer, return its corresponding column title as appear in an Excel sheet.

For example:

   1 -> A
   2 -> B
   3 -> C
   ...
   26 -> Z
   27 -> AA
   28 -> AB

思路：

记第$k$个字母为$x_k$，则 $$ n=a_{k}\times 26^{k-1}+\cdots+a_2\times 26 +a_1 $$ 其中$a_j=ord(x_j)-64$，即$x_j=chr(a_j+64)$，并且$1\leq a_j\leq 26.$

因此 $$ a_1=(n-1)\%26+1, x_1=chr(a_1+64) $$ 再递归处理 $(n-1)//26$.

class Solution:
    def convertToTitle(self, n):
        """
        :type n: int
        :rtype: str
        """
        ## base case
        if n<=26:
            return chr(n+64)

        a, n = (n-1)%26, (n-1)//26
        return "{}{}".format(self.convertToTitle(n), chr(a+65))
