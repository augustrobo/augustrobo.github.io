Python Type Conversion

2018/01/01

Categories: Python Tags: pythonBasics

  • Types
  • Explicit Data Type Conversion

Types

  • Variables in Python are dynamically-typed: declared without an explicit type.
  • objects have a type, so Python knows the type of a variable.
  • No char in Python! Both ' and " create string literals
type(1)
# <class 'int'>
type(1.0)
# <class 'float'>
type(2/1)
# <class 'float'>
type("hello")
# <class 'str'>
type('a')
# <class 'str'>
type(True)
# <class 'bool'>
type(None)
# <class 'NoneType'>
type(int)
# <class 'type'>
type(type(int))
# <class 'type'>
type(print)
# <class 'builtin_function_or_method'>
  • bool is a subtype of int, where True == 1 and False == 0
True == 1
# True
False == 0
# True
True == 2
# False
True or False
# True (short-circuits)

Explicit Data Type Conversion

str(333)
# '333'
int("33")
# 33
int("33.3")
# ValueError
float("33.3")
# 33.3
float("2")
# 2.0
float("1/2")
# ValueError
float(1/2)
# 0.5
