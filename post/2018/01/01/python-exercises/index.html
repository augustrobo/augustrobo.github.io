Greatest Common Divisor 最大公因数

Euclid’s algorithm:

def gcd(p, q):
    '''Compute the greatest common divisor of 2 nonnegative integers
    '''
    if q==0:
        return p
    else:
        return gcd(q, p%q)

Two integers $p,q\geq 0$,

(1) obviously, $gcd(p, 0)=p$

(2) $gcd(p,q)=gcd(q,r)$, where $r=p\%q$

Proof of statement (2):

Let $a=gcd(p,q)$ and $b=gcd(q,r)$, then $$ \left. \begin{align} a|p, a|q \Rightarrow a|r \Rightarrow a\leq gcd(q,r)=b\nonumber \newline b|q, b|r \Rightarrow b|p \Rightarrow b\leq gcd(p,q)=a \nonumber \end{align} \right\} \Rightarrow a=b $$ Proof of termination of Euclid’s algo:

If $p<q$, then next step $r=p$. Therefore assume $p>q>0$ $$ \begin{cases} q<p/2 &\Rightarrow r<q<p/2\newline q\geq p/2 &\Rightarrow r=p-q\leq p/2 \end{cases} $$ Therefore, # of recursions $\sim \log_2p+\log_2q$.

Power

def pow(a,n):
    '''
    Calculate the power a^n
    a: base, positive real number
    n: exponent, positive integer
    '''
    if n == 1:
        return a

    result = pow(a*a, n//2)
    if n%2 == 0:
        return result
    else:
        return a*result

$$ T(n)=T\left(\frac{n}{2}\right)+O(1) \Rightarrow a=1=2^0=b^d \text{(case 1)} \Rightarrow T(n)=O(\log n) $$
